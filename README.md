**Smart Resume Filter & Career Recommendation System**

🚀 Overview

A unified, cost-free platform that bridges the gap between recruiters and talent using statistical matching and Generative AI.

🛠️ Tech Stack
Backend: Python, FastAPI, Uvicorn.

AI/NLP: Groq Inference Engine (Llama 3), spaCy, pdfplumber.

Frontend: React.js, Chart.js.
