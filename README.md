**Smart Resume Filter & Career Recommendation System**

🚀 Overview

A unified, cost-free platform that bridges the gap between recruiters and talent using statistical matching and Generative AI.

🛠️ Tech Stack
Backend: Python, FastAPI, Uvicorn.

AI/NLP: Groq Inference Engine (Llama 3), spaCy, pdfplumber.

Frontend: React.js, Chart.js.

🧠 How it Works

The system extracts skills from resumes and job descriptions to calculate a match score based on Cosine Similarity:

✨ Key Features

Bulk Ranking: Rank multiple candidates instantly.

AI HR Assistant: Ask specific questions about a candidate's experience.

Personalized Roadmaps: Actionable feedback for job seekers to fix skill gaps
